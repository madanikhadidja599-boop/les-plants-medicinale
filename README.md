# les-plants-medicinale
Une plante médicinale est une plante utilisée pour ses propriétés thérapeutiques afin de soulager divers maux. Elles peuvent être utilisées pour traiter des problèmes de santé comme les troubles digestifs (cardamome, absinthe), les maux de gorge et la toux (plantain, guimauve) ou encore les douleurs et l'inflammation (arnica, chanvre).
